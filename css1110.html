What I learned in CSS1110

To start off with, there needs to be a point of a distinct differences between Hypertext Markup Language (HTML) and Cascading Style Sheet (CSS).

  • HTML is a language that allows users to structure their page in the place they want the text and tables and lists to be.
  • CSS is a language that allows users to style their HTML, being able to add color, padding, margins, and differenct fonts are important in making a page made out of HTML to stand out.
